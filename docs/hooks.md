# latex-unicode-hooks

## `unicode-base`

Checks that Unicode is used for basic $\LaTeX$ symbols, like `‚àá` instead of `\nabla`.

## `unicode-greek`

Checks that Unicode Greek letters are used.

## `unicode-subscripts`

Checks that Unicode subscripts are used.

## `unicode-superscripts`

Checks that Unicode superscripts are used.

## `unicode-integrals`

Checks that Unicode integrals are used.

## `unicode-quotes`

Checks that Unicode quotes are used.

## `unicode-transpose`

Checks that `ûÅÄ` (modifier letter cyrillic small te (U+1E040)) is used for transpose.

## `unicode-hermitian`

Checks that `·µ∏` (modifier letter cyrillic en (U+1D78)) is used for hermitian.
